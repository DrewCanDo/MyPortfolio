FireBorn Mobile App

FireBorn Mobile Application (Apr 2023)
Download Project (ZIP) Github Repository
  • **Overview**:
    • A mobile app built to provide users with accessible, visual information about meteors and NASA data.
    • Developed in Android Studio using Java, with Firebase for user authentication and secure data storage.
  • **Course Context**:
    • Created as a project for an Android development course with a focus on integrating APIs and building a smooth user experience.
    • Emphasized using Firebase for backend services and Google Maps for location-based features.
  • **Key Features**:
    • Google Maps integration to display meteor data and locations on Earth.
    • User profile customization with image upload and name personalization, utilizing Picasso for image resizing.
  • **Design/Technical Decisions**:
    • Chose Firebase for secure user authentication and to store user profile data safely.
    • Integrated the NASA API to convert complex data into pie charts and histograms for easier user comprehension.
  • **Challenges & Learning**:
    • Learned to handle complex data visualization, converting NASA data into readable charts and graphs.
    • Overcame performance issues by optimizing app speed using threads for multitasking and Picasso for efficient image handling.
  • **Future Enhancements**:
    • Plan to expand meteor data sources and provide users with more in-depth information on meteor impacts.
    • Interested in adding real-time notifications for upcoming meteor showers or events based on user location.
