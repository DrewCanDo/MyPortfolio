Book Corner Website (2023)
Download Project (ZIP) Github Repository
  • **Overview**:
    • A web application for a book club where users can organize and RSVP for meetups.
    • Built with Express.js, Node.js, MongoDB, and dynamic rendering for a smooth user experience.
  • **Course Context**:
    • Developed as part of a web development course focusing on full-stack JavaScript development.
    • Emphasized the use of Express and MongoDB for backend logic and data storage.
  • **Key Features**:
    • User authentication for account creation, login, and personalized profiles.
    • Meetup creation, RSVP management, and a simple messaging system for event coordination.
  • **Design/Technical Decisions**:
    • Used Express.js for building RESTful APIs and server-side logic.
    • Integrated dynamic loading for the header and footer to enhance the site’s maintainability and user experience.
  • **Challenges & Learning**:
    • Learned to manage dynamic content loading to optimize page performance and scalability.
    • Overcame challenges in working with MongoDB and Mongoose to efficiently manage data relationships.
  • **Future Enhancements**:
    • Plan to implement user reviews for books and events to build community feedback.
    • Interested in adding more advanced features like real-time chat and event notifications.
